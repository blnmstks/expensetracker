# Исправление: Мобильный навбар не виден на странице Аналитики

## 🐛 Проблема
На мобильных устройствах нижний навбар не был виден при переходе на страницу Аналитики.

## 🔍 Причины
1. **Низкий z-index**: Навбар имел `z-index: 100`, что недостаточно
2. **Sticky элементы в таблице**: Таблица с sticky колонками могла перекрывать навбар
3. **Overflow проблемы**: Широкая таблица могла вызывать горизонтальный скролл
4. **Лишние отступы**: Компонент Analytics имел padding, который мешал на мобильных

## ✅ Исправления

### 1. Увеличен z-index для навбара
**Файл:** `src/styles/antd-custom.css`

```css
.mobile-bottom-nav {
  z-index: 1000; /* было 100 */
}
```

### 2. Увеличен z-index для мобильного хедера
**Файл:** `src/components/layout/MainLayout.tsx`

```tsx
<Header style={{ zIndex: 1000 /* было 100 */ }}>
```

### 3. Адаптивные отступы в Analytics
**Файл:** `src/components/Analytics.tsx`

```tsx
// Было:
<div className="max-w-full mx-auto space-y-6 p-6">

// Стало:
<div className="max-w-full mx-auto space-y-6 md:p-6">
```

Теперь padding применяется только на десктопе (md и выше).

### 4. Адаптивный заголовок и селектор
**Файл:** `src/components/Analytics.tsx`

```tsx
// Было:
<h2 className="text-3xl ...">

// Стало:
<h2 className="text-2xl md:text-3xl ...">

// И селектор валюты:
<div className="w-full sm:w-48">
```

### 5. Предотвращение горизонтального скролла
**Файл:** `src/index.css`

```css
body, html {
  overflow-x: hidden;
  width: 100%;
  position: relative;
}

#root {
  overflow-x: hidden;
  width: 100%;
}
```

### 6. Дополнительные мобильные стили
**Файл:** `src/styles/antd-custom.css`

```css
@media (max-width: 768px) {
  /* Sticky элементы не должны перекрывать навбар */
  .sticky {
    z-index: 10 !important;
  }
  
  /* Убеждаемся, что контент не выходит за границы */
  body {
    overflow-x: hidden;
  }
}
```

## 📊 Иерархия z-index

```
┌─────────────────────────────────────┐
│  z-index: 1050                      │  Ant Design Dropdown, Popover
├─────────────────────────────────────┤
│  z-index: 1000                      │  Мобильный навбар & хедер
├─────────────────────────────────────┤
│  z-index: 10                        │  Sticky элементы в таблицах
├─────────────────────────────────────┤
│  z-index: 1                         │  Карточки, таблицы
└─────────────────────────────────────┘
```

## 📱 Адаптивность

### Мобильные устройства (<768px):
- Верхний хедер: `z-index: 1000`
- Нижний навбар: `z-index: 1000`
- Отступы контента: `padding-bottom: 80px`
- Без лишних padding на компонентах

### Планшеты и десктопы (≥768px):
- Боковой сайдбар всегда виден
- Нормальные отступы
- Нет нижнего навбара
